Requirements

System requirements and dependencies

Core Requirements

  • Architecture: x86_64, ARM64, ARMv7
  • RAM: Minimum 2GB, Recommended 4GB+
  • Storage: 5GB free space for VM + containers
  • OS: Linux, macOS, Windows, Android (Termux)

Dependencies

Linux/Android:

proot-distro qemu-system-x86_64 qemu-utils wget curl openssh bash

macOS:

qemu wget curl openssh bash

Windows:

qemu wget curl openssh bash (via WSL/MSYS2)

Optional

  • Docker: Alternative container runtime
  • Podman: Alternative container runtime
  • VNC Viewer: For graphical VM access
  • virt-manager: GUI VM management

Quick Dependency Installation

Termux (Android)

pkg update && pkg install proot-distro qemu-system-x86-64 qemu-utils wget curl openssh

Ubuntu/Debian

sudo apt update && sudo apt install qemu-system-x86 qemu-utils wget curl openssh-server

Arch Linux

sudo pacman -S qemu-desktop wget curl openssh

macOS

brew install qemu wget curl openssh
